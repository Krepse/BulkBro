🔧 Database Migration

Fix Warmup Timer Persistence Bug

⚠️ Viktig: Denne migrasjonen legger til en kolonne i databasen for å fikse problemet med forsvinnende oppvarmingstimer.

📋 Hva migrasjonen gjør:

Legger til start_time kolonne i sett tabellen
Dette gjør at timer-data blir lagret permanent i databasen
Eksisterende data påvirkes ikke (trygg operasjon)

💻 SQL som kjøres:

ALTER TABLE public.sett ADD COLUMN IF NOT EXISTS start_time TIMESTAMPTZ;
